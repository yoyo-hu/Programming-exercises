# Ant

## Description

一根长度为L厘米的木棒上有N只蚂蚁，每只蚂蚁要么向左走，要么向右走，速度为1厘米/秒。当两只蚂蚁相撞时，他们会同时掉头（掉头时间不计）给出每只蚂蚁距离木棒左端的距离，问多少秒后，刚好所有蚂蚁都从木棒上掉下来。

**N** 和 **L** 均不超过1000

输入第一行两个整数，分别是**N**和**L**

接下来**N**行，每行先是一个字符，`L`或`R`，代表向左还是向右，然后是一个整数`x`，代表这个蚂蚁距离木棒**左端**的距离。

### 样例输入：

```
4 10
R 1
R 5
L 3
R 9

```

### 样例输出：

```
9

```

### Hint

假设你在远处观察两只蚂蚁相向而行，他们相撞后掉头，和直接穿过有什么区别？
