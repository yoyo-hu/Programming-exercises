# 字符串2

任何一个正整数都可以用2的幂次方表示。例如：

$137=2^7+2^3+2^0$

同时约定次方用括号来表示，即$a^b$可表示为$a(b)$。由此可知，137可表示为：2(7)+2(3)+2(0)

进一步：7=2(2)+2+2(0)，3=2+2(0)

所以最后137可表示为：2(2(2)+2+2(0))+2(2+2(0))+2(0)

### 输入

一个正整数n(n<=20000)

### 输出

一行，符合约定的n的0,2表示（在表示中不能有空格）

### 样例输入

```
137
```

### 样例输出

```
2(2(2)+2+2(0))+2(2+2(0))+2(0)
```
